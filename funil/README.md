# Funil Sopro Life v1

Funil estático para GitHub Pages.

## O que ele faz
- Perguntas com ramificação simples
- Barra de progresso
- Coleta nome, WhatsApp, bairro e horário
- Gera mensagem pronta para o WhatsApp da Sopro Life
- Pode salvar lead em Google Sheets via Apps Script se você preencher `LEAD_ENDPOINT` no `app.js`

## Como publicar no site atual
1. Copie a pasta `funil-soprolife-v1` para dentro do repositório do site.
2. Renomeie a pasta para `funil` se quiser.
3. Faça commit e push.
4. Acesse: `https://soprolife.com.br/funil/`
